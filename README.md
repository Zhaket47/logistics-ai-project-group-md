### Project Overview
This system is an end-to-end intelligent analytics solution for logistics. It uses Machine Learning to forecast demand, predict delivery delays, and optimize inventory levels.

### Dataset
* **Source:** [Supply Chain Analysis on Kaggle](https://www.kaggle.com/datasets/harshsingh2209/supply-chain-analysis/data)
* **Description:** Real-world logistics data including products, shipping costs, and supplier performance.

### Tools & Libraries
* **Python:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn.
* **BI:** Power BI Desktop.

### Key Business Insights
1. **Inventory Crisis:** 81 items identified in "Understock" status; AI-driven reorder points are required.
2. **Bottleneck Identification:** 28 high-risk routes found where costs and delays exceed the corporate average.
3. **Sales Driver:** AI models show that product availability has a 20x higher impact on sales than price.
